# Vue.js & Laravel Chat App

A modern real-time chat application designed for direct 1-on-1 conversations. This project leverages the reactive power of Vue.js for the frontend and the robust backend capabilities of Laravel. It is built for seamless user experiences, reliable authentication, and fast, secure messaging.

---

## Features

- **Direct 1-1 Chat**: Instantly chat in real time with any friend in your contact list.
- **User Authentication**: Secure login and registration system powered by JWT.
- **Friends Management**: Add, remove, and manage your friends for easy communication.

## Tech Stack

- **Frontend**: [Vue.js](https://vuejs.org/)  
  - Modern, reactive JavaScript framework constructed for powerful UIs.
- **Backend**: [Laravel](https://laravel.com/)  
  - Secure, feature-rich PHP framework designed for rapid development.
- **Real-Time**:  
  - Utilizes Laravel Broadcasting and WebSockets for instant message delivery.

---

## Setup

1. **Clone the repository**
2. **Install dependencies** for both frontend and backend
3. **Configure your .env** files (API keys, database, broadcast driver, etc.)
4. **Run database migrations**
5. **Start both backend and frontend servers**
6. **Open the app in your browser and start chatting!**
